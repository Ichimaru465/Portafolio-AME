## 📌 Descripción

Este repositorio contiene el **portafolio académico del curso de Desarrollo de Aplicaciones Web**, donde se evidencian los trabajos realizados semana a semana durante el ciclo.

El portafolio refleja el aprendizaje progresivo en tecnologías **frontend y backend**, incluyendo el desarrollo de aplicaciones web modernas, consumo de APIs y despliegue de sistemas.

* Documentar el progreso del curso semanalmente

## ▶️ Ejecución en Local

Para visualizar el proyecto en local:

1. Clonar el repositorio:

```
git clone https://github.com/Ichimaru465/Portafolio-AME.git
```

2. Abrir la carpeta en un editor de código

3. Ejecutar con servidor local (recomendado):

* Usar Live Server en VS Code
* O ejecutar:

```
python -m http.server 8000
```

4. Abrir en navegador:

```
http://localhost:8000
```
